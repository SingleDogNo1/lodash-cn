# shuffle

::: tip 语法

_.shuffle(collection)

:::

## 描述

创建一个被打乱值的集合

## 参数

|    参数    |      类型      | 默认值 |     说明     |
| :--------: | :------------: | :----: | :----------: |
| collection | Array / Object |   -    | 要打乱的集合 |

## 返回值

+ Array: 返回打乱顺序的新数组

## 示例

```js
_.shuffle([1, 2, 3, 4]);
// => [4, 1, 3, 2]
```
