# isArguments

::: tip 语法

_.isArguments(value)

:::

## 描述

检查 `value` 是否是一个类 `arguments` 对象。

## 参数

|  参数  | 类型  | 默认值 |       说明       |
| :----: | :---: | :----: | :--------------: |
| value  |  any  |   -    |    要检查的值    |

## 返回值

+ boolean: 如果`value`是一个`arguments`对象，返回 `true`，否则 `false`。

## 示例

```js
_.isArguments(function() { return arguments; }());
// => true

_.isArguments([1, 2, 3]);
// => false
```
