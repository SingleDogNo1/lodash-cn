# first

::: tip 语法

_.head(array)
_.first(array)

:::

## 描述

获取数组 `array` 的第一个元素。

## 别名

`_.first`

## 参数

| 参数  | 类型  | 默认值 |     说明     |
| :---: | :---: | :----: | :----------: |
| array | Array |   -    | 要查询的数组 |

## 返回值

+ any(any): 返回数组的第一个元素。

## 示例

```js
_.head([1, 2, 3]);
// => 1

_.head([]);
// => undefined
```
