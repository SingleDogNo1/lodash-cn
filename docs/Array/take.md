# take

::: tip 语法

_.take(array, [n=1])

:::

## 描述

创建一个数组切片，从array数组的起始元素开始提取n个元素

## 参数

| 参数  |  类型  | 默认值 |       说明       |
| :---: | :----: | :----: | :--------------: |
| array | Array  |   -    |   要检索的数组   |
|   n   | number |   1    | 要提取的元素个数 |

## 返回值

+ array(Array): 返回从开始截取到第n个元素的数组切片

## 示例

```js
_.take([1, 2, 3]);
// => [1]

_.take([1, 2, 3], 2);
// => [1, 2]

_.take([1, 2, 3], 5);
// => [1, 2, 3]

_.take([1, 2, 3], 0);
// => []
```
