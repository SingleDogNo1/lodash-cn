# initial

::: tip 语法

_.initial(array)

:::

## 描述

获取数组`array`中除了最后一个元素之外的所有元素。

## 参数

|   参数    |  类型  | 默认值 |      说明      |
| :-------: | :----: | :----: | :------------: |
|   array   | Array  |   -    | 需要查找的数组 |

## 返回值

+ array(Array): 返回截取后的数组array

## 示例

```js
_.initial([1, 2, 3]);
// => [1, 2]
```
