# drop

::: tip 语法

_.drop(array, [n=1])

:::

## 描述

创建一个切片数组，去除array前面的n个元素。（n默认值为1。）

## 参数

|     参数     |   类型   | 默认值 |      说明      |
| :----------: | :------: | :----: | :------------: |
|    array     |  Array   |   -    | 需要处理的数组 |
|   [n=1]   | number |   1    |   要删除的个数   |

## 返回值

+ array (Array): 返回`array`的剩余切片

## 示例

```js
_.drop([1, 2, 3]);
// => [2, 3]

_.drop([1, 2, 3], 2);
// => [3]

_.drop([1, 2, 3], 5);
// => []

_.drop([1, 2, 3], 0);
// => [1, 2, 3]
```
