# zip

::: tip 语法

_.zip([arrays])

:::

## 描述

创建一个分组元素的数组，数组的第一个元素包含所有给定数组的第一个元素，数组的第二个元素包含所有给定数组的第二个元素，以此类推。

## 参数

|    参数    |   类型   | 默认值 |     说明     |
| :--------: | :------: | :----: | :----------: |
|   array    | ...Array |   -    | 要处理的数组 |

## 返回值

+ array(Array): 返回分组元素的新数组

## 示例

```js
_.zip(['fred', 'barney'], [30, 40], [true, false]);
// => [['fred', 30, true], ['barney', 40, false]]
```
