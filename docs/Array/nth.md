# nth

::: tip 语法

_.nth(array, [n=0])

:::

## 描述

获取`array`数组的第n个元素。如果n为负数，则返回从数组结尾开始的第n个元素。

## 参数

| 参数  |  类型  | 默认值 |        说明        |
| :---: | :----: | :----: | :----------------: |
| array | Array  |   -    |    要检索的数组    |
|   n   | number |   0    | 要返回元素的索引值 |

## 返回值

+ any(any): 返回array数组的第n个元素

## 示例

```js
var array = ['a', 'b', 'c', 'd'];

_.nth(array, 1);
// => 'b'

_.nth(array, -2);
// => 'c';
```
